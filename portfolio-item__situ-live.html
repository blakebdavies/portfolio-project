Case Study: Situ Live

Built with HTML, CSS, and Javascript

This is a Vanilla JavaScript project focused on creating a simple, user-friendly carousel feature displaying the Situ Live brand. My end goal of this project was to create a fully functional carousel of 4 brand images that included a title and description overlay. I created this feature to also wanted to build on my HTML, CSS, and Javascript skillset and to showcase the ways in which Situ Live could repurpose their content and refresh their website.

To accommplish this project, I first started by creating a wireframe using InVisionApp to map out the basic design. I then created a prototype of this feature using Figma to ensure each element aligned with the Situ Live brand guidelines.

Throughout this project, I strengthened my understanding of Javascript by implementing new concepts like appendChild(), writing functions, and learning CSS features like translate()and transform(). To enhance the UI/UX, I also engineered dot indicators and time scrolling to improve the design and functionality.
