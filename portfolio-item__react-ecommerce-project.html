Case Study: Brightspot Mobile

React E-Commerce Website

This is a React project focused on creating a simple, user-friendly design with complete Paypal purchasing functionality. I built this fully-functioning, e-commerce website, gaining and developing my skills and understanding in Styled Components, APIs, responsive design, and environment variables for enhanced security.

My end goal of this project was to create a fully functioning e-commerce mobile phone store that allows users to add/remove products to their cart with the total payment automatically updating. I also wanted to take this project a step further by allowing users to 'purchase' the products using the React Paypal Express Checkout installed via npm.
